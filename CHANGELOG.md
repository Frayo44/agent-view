# Changelog

## [Unreleased]

### Added
- **Recents** (`o`): Quickly recreate sessions from recent configurations. Sessions are automatically saved when created (including their group), and you can press `o` to open the recents dialog and start a new session with the same path, tool, and group. If the group doesn't exist, it will be created automatically. Claude sessions always start fresh (no resume).
